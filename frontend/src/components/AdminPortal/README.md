# Admin Portal Control Deck Components

This folder contains the modular subcomponents for the rebuilt RailLuxury admin panel landing portal:

1. `PortalHeader.jsx` - Top banner with server node latency telemetry.
2. `SystemMetrics.jsx` - CPU & Memory sparklines and visual metrics dashboard.
3. `ServiceControlGrid.jsx` - Service controls and console override switches.
4. `LiveLogTerminal.jsx` - Command-line log terminal supporting custom commands.
5. `AccessHandshake.jsx` - Admin portal credential authentication & security PIN-pad keypad.
6. `SecurityAlertBanner.jsx` - Warning threat indicator panel.
